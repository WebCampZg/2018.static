Fearless Concurrency with JavaScript and Beyond

Super-charge your single-page application with proper concurrency! A thorough review of both popular and unorthodox approaches.

The conventional JavaScript environment is that of single-threaded cooperative multitasking which in some applications is approaching its limits. Given the obvious upper boundary of how far single-threaded code optimisation can go, concurrency is almost a requirement for today’s applications.

Within this talk, I shall…

1/ demonstrate the benefits of advancing beyond single-threaded JavaScript programs; 2/ analyse various concurrency designs that fit within the browser; and 3/ present implementations that can be repurposed by the audience today.

The audience is expected to benefit from the talk by:

1/ gaining new insights and new cross-cutting vantage points towards performance problem analysis and solution design; 2/ learning new techniques that can be used for special situations that they may run into.

> Skill level: advanced
