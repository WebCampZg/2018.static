Fluid typography

We scale down interfaces fluidly, yet often rely only on breakpoints for responsive typography. Let's make letters fluid, as they should be!

Why we keep letters bound between rigid breakpoints when all those screen sizes await?

We will learn to implement fluid typography best practices by using browser-based calculations to get that perfectly scalable letters we all want, but maybe don't know it yet. For some time now, modern browsers understand responsive units and with some clever calculations, it is possible to scale letters up and down and nailing down the perfect font size for every possible resolution.
